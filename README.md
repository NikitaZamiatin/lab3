# Лабораторная работа №3 - Шахматы
Пользователь вводит координаты двух полей шахматной доски, а программа проверяет, являются ли цвета выбранных полей одинаковыми, а также угрожает ли фигура, стоящая на первом поле второму, если нет, то выясняет, можно ли добраться этой фигурой от первого до второго поля за два хода.
# Среда разработки
Язык программирования: Python.
Среда разработки: Visual Studio Code.
# Инструкция по работе
По запросу программы необходимо ввести сначала две цифры, обозначающие координаты первого поля, а далее две цифры, обозначающие координаты второго поля. После следует ввести одно слово из представленных с маленькой буквы.
